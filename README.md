# PAW-CARE Pet Clinic Grooming MiniApps Python CLI
PAW-CARE is a Command Line Interface (CLI) based pet clinic and grooming management system built using Python. The application is designed to support and streamline daily operational activities in a pet clinic and grooming business through structured data management and modular programming.
PAW-CARE/

-----

## Objectives
- Develop a CLI-based management system for pet clinic and grooming services
- Manage patient registration and records
- Organize available services and pricing
- Assign staff to patients
- Process payments accurately
- Improve operational efficiency
- Reduce manual processes and data inconsistencies

-----

## Features
- User Authentication (Admin and Staff roles)
- Patient Registration
- Patient Record Management
- Service Management
- Staff Management
- Staff Assignment
- Payment and Checkout Processing
